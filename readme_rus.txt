Скрипт считывает из com-порта штрих-код, после чего эмулирует нажатие каждой считанной цифры и в конце так же эмуляция клавиши <Enter>.

Данный скрипт был создан для прайс-чекера(raspberry pi который работает в режиме киоска). 

Схема:

Запущен браузер. 
В браузере есть скрытое поле для ввода, в нём установлен фокус ввода.
Скрипт считывает штрих-ход, эмулирует нажатие каждой считанной цифры, и нажимает <Enter>.
Браузер отправляет штрих-код на сервер.
Сервер отвечает ценой товара.
